🎯 ¿Qué es TicQuiz IA?

🤖

Chatbots con IA

4 profesores virtuales con personalidades únicas: Shulian (Programación), Darío (Hardware), Pau (TIMI) e Ivo (Redes). Cada uno especializado en su materia.

📝

Quizzes Inteligentes

Generación automática de preguntas de opción múltiple a partir de cualquier contenido educativo. Configurable de 1 a 20 preguntas.

📄

Subida de Archivos

Soporte para .txt, .pdf e imágenes con OCR. El contenido se extrae automáticamente para generar quizzes o alimentar los chatbots.

🎮

Juegos Educativos

Juego de Memoria, Ahorcado y Sopa de Letras. Todos generados por IA con el contenido de la materia.

👨‍🏫 Panel del Docente

📋 Contenido

  • Subir archivos (.txt, .pdf, imágenes)
  • Importar texto desde URLs (Google Docs, web)
  • Guardar contenido temático

📚 Contexto Compartido

  • Material del curso que los chatbots usan como referencia
  • Importar desde URL o pegar texto manualmente
  • Indicador visible para alumnos cuando hay contexto activo

📊 Resultados

  • Ver resultados de todos los alumnos
  • Notificaciones de quizzes completados y vencidos
  • Detalle de respuestas correctas/incorrectas

❓ Quizzes

  • Generar quizzes con IA desde contenido propio
  • Seleccionar contenido guardado como base
  • Configurar cantidad de preguntas (1-20)
  • Asignar Curso (3ro, 4to, 5to) — obligatorio
  • Asignar Materia (Hardware, Software, Redes, TIMI) — obligatorio
  • Fecha límite de entrega opcional
  • Publicar para todos los alumnos del curso

💬 Chatbots

  • Conversar con los 4 profesores IA
  • Chats separados por profesor
  • Historial persistente por usuario
  • Soporte de Markdown en respuestas

🎓 Panel del Alumno

💬 Chatbots

  • Conversar con profesores IA por materia
  • Indicador de contexto activo del curso
  • El chatbot usa el material del docente como referencia
  • Detección automática de URLs en mensajes

❓ Quizzes

  • Filtrar por Curso (3ro, 4to, 5to)
  • Filtrar por Materia (Hardware, Software, Redes, TIMI)
  • Resolver quizzes publicados por el docente
  • Generar quizzes de práctica propios
  • Resultados enviados automáticamente al docente

🎮 Juegos

  • Memoria: Emparejar conceptos con definiciones
  • Ahorcado: Adivinar palabras con pistas
  • Sopa de Letras: Encontrar palabras en la grilla
  • Contenido generado por IA según la materia

🎬 Reels Educativos

  • Formato vertical tipo YouTube Shorts / TikTok
  • 8 reels generados por IA con el contenido del curso
  • Audio narrado automáticamente (Web Speech API)
  • Diferentes esquemas de color por reel
  • Scroll vertical con puntos de navegación

📊 Mis Resultados

  • Historial de quizzes completados
  • Puntaje y porcentaje por intento
  • Revisión detallada pregunta por pregunta

⚙️ Características Técnicas

🧠

Groq AI

Modelo llama-3.1-8b-instant para generación de quizzes, respuestas de chatbots, juegos y reels. Rápido y eficiente.

🔐

Autenticación

Registro con códigos especiales (DOCENTE2024 / ALUMNO2024). Perfiles con curso seleccionable (3ro/4to/5to).

💾

Persistencia

Datos guardados en archivos JSON: usuarios, quizzes, resultados, historial de chat, notificaciones y contexto.

🔍

OCR

Extracción de texto de imágenes mediante Tesseract.js. Soporte para .txt, .pdf y formatos de imagen.

🎨

Markdown

Los chatbots responden con formato Markdown: negrita, listas, bloques de código con resaltado de sintaxis, enlaces.

🔔

Notificaciones

Sistema de notificaciones para quizzes completados, vencidos y nuevas tareas con plazo límite.

🖥️ Tecnologías Utilizadas

⚛️

Next.js 14

Framework React con App Router. API routes para todas las funcionalidades del backend.

🎨

Tailwind CSS v4

Estilos utilitarios con configuración vía CSS (@import "tailwindcss"). Diseño responsive.

🧠

Groq SDK

API de inferencia rápida con modelo llama-3.1-8b-instant para todas las funciones de IA.

📝

react-markdown

Renderizado de Markdown con remark-gfm para tablas, listas y formato en chats.

🔍

Tesseract.js

Reconocimiento óptico de caracteres (OCR) para extraer texto de imágenes subidas.

📄

pdf-parse

Extracción de texto desde archivos PDF para procesamiento educativo.

JavaScript React Node.js Web Speech API JSON Persistence ESM Modules
